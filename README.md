# InstantSEO

Prestashop module to easily add Structured Data compliant with Schema.org to your shop

It handles the following:

- WebSite
- WebPage
- SiteNavigationElement
- Breadcrumbs
- Organization (with Socials)
- LocalBusiness

## Prerequisites

- Prestashop versions: 1.7.6 - 1.7.7.6
